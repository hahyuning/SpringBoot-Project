# SpringBoot-project

Java11, Spring Boot, OAuth2, JWT, JPA, MySQL

- 로그인 
- 세션 관리
- 마이페이지

- 사진 첨부
- 익명 글 작성
- 관련 아이템 등록 (이미지 인식)
- 좋아요
- 댓글/대댓글
